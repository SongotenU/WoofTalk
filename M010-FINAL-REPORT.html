iOS App Store Submission (Phase 59)

Requires Xcode GUI Archive workflow with valid code signing identity and distribution certificate. Cannot be automated in CI without App Store Connect API key setup. Must be run manually on a macOS machine with the appropriate provisioning profiles.

!

Google Play Store Upload (Phase 60)

Requires manual upload to Google Play Console. Store listing (screenshots, description, privacy policy URL) must be configured. AAB artifact must be built and signed with the release key before upload.

⚠

Android AGP Classloader Conflict (Phase 56)

AGP 8.2.2 to 8.7.0 upgrade fixed compatibility issues, and Gradle wrapper updated to 8.9 with Java 21. However, a Gradle 8.x AGP classloader bug persists on cached build environments. Workaround: clean build (gradlew clean) or use a fresh checkout. The Android build itself is now structurally correct.
