# GSD Skill Preferences

## Active Skills

- **debug-like-expert**: Deep analysis debugging mode for complex issues. Activates methodical investigation protocol with evidence gathering, hypothesis testing, and rigorous verification. Use when standard troubleshooting fails or when issues require systematic root cause analysis.
- **frontend-design**: Create distinctive, production-grade frontend interfaces with high design quality. Use this skill when the user asks to build web components, pages, artifacts, posters, or applications (examples include websites, landing pages, dashboards, React components, HTML/CSS layouts, or when styling/beautifying any web UI). Generates creative, polished code and UI design that avoids generic AI aesthetics.
- **swiftui**: SwiftUI apps from scratch through App Store. Full lifecycle - create, debug, test, optimize, ship.

## Skill Selection Guidelines

- Use **swiftui** for SwiftUI-specific tasks and iOS app development
- Use **frontend-design** for UI/UX design and frontend development
- Use **debug-like-expert** for complex debugging and troubleshooting
- Default to standard iOS development practices unless otherwise specified

## Skill Integration

- Skills should be loaded based on task requirements
- Skill preferences should guide but not override required plan formatting
- Use the lightest sufficient skill first
- Combine skills when tasks span multiple domains
