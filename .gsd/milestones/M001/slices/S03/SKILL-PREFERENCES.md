# GSD Skill Preferences

## Frontend Design

- **UI Framework**: UIKit (per D007 decision)
- **Design Style**: Clean, professional iOS design with smooth animations
- **Color Scheme**: Dog-friendly colors (blues, greens, neutral tones)
- **Accessibility**: High contrast, large touch targets, VoiceOver support
- **Animation**: Smooth transitions, latency indicators, loading states

## Swift/iOS Development

- **Architecture**: MVC with proper separation of concerns
- **Performance**: <2 second latency requirement must be maintained
- **Offline**: SQLite database for vocabulary storage (per D008)
- **Audio**: AVFoundation for audio processing (per D010)
- **Testing**: XCTest for unit and UI testing
- **App Store**: Full compliance with Apple guidelines (per D009)
