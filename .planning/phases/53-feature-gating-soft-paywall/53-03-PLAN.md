# Wave 3: Feature Gating for Premium Features

## Objective
Restrict AI translation, community phrase contribution, and export/share features to premium users.
Show upgrade prompt (non-blocking modal/toast) when free users attempt these actions.
Display lock icons on disabled premium features in UI.
Ensure upgrade prompt links directly to paywall for seamless conversion.

## Tasks

### AI Translation Feature Gating (All Platforms)
- [ ] Before initiating AI translation, check EntitlementManager.isPremium
- [ ] If !isPremium, show non-blocking upgrade prompt (do not initiate AI translation)
- [ ] If isPremium, proceed with AI translation
- [ ] Update AI translation toggle/button to show lock icon when !isPremium
- [ ] Add tooltip/hint indicating AI translation is premium feature

### Community Phrase Contribution Gating (All Platforms)
- [ ] Before allowing phrase submission, check EntitlementManager.isPremium
- [ ] If !isPremium, show non-blocking upgrade prompt (do not allow submission)
- [ ] If isPremium, allow phrase submission
- [ ] Update contribution button/UI to show lock icon when !isPremium
- [ ] Add tooltip/hint indicating community contribution is premium feature

### Export/Share Feature Gating (All Platforms)
- [ ] Before allowing export/share, check EntitlementManager.isPremium
- [ ] If !isPremium, show non-blocking upgrade prompt (do not allow export/share)
- [ ] If isPremium, allow export/share
- [ ] Update export/share button/UI to show lock icon when !isPremium
- [ ] Add tooltip/hint indicating export/share is premium feature

### Upgrade Prompt Implementation (All Platforms)
- [ ] Create reusable upgrade prompt component (modal/bottom sheet/toast)
- [ ] Prompt includes: message about premium feature access, "Upgrade" button, dismiss option
- [ ] "Upgrade" button navigates to paywall entry point (from Phase 52)
- [ ] Prompt is non-blocking - user can dismiss and continue using free features
- [ ] Ensure consistent styling across platforms

## Dependencies
- Wave 1: Entitlement Manager Extension (provides isPremium property)
- Wave 2: Translation Gating (establishes upgrade prompt pattern)
- Phase 52: Paywall UI (provides paywall entry points for upgrade prompt navigation)

## Success Criteria
- AI translation restricted to premium users — free users see upgrade prompt when attempting
- Community phrase contribution restricted to premium users — free users see upgrade prompt when attempting
- Export/share restricted to premium users — free users see upgrade prompt when attempting
- Premium features display lock icons in UI when user is on free tier
- Upgrade prompt is non-blocking and links directly to paywall for seamless conversion
- Premium users have unrestricted access to all features
