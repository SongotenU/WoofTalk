# Plan 54-01: iOS & Watch — Cross-Platform Sync + Restore + Manage Subscription

**Phase**: 54 — Cross-Platform Sync & Admin
**Scope**: iOS app + WatchKit extension
**Depends on**: Phase 50 (RevenueCat SDK), Phase 52 (Paywall UI)

## Goal

Ensure iOS subscription entitlements sync to Watch instantly via Watch Connectivity, restore purchases works, and users can manage subscriptions via App Store link.

## Tasks

### 1. Add Restore Purchases to SettingsViewController
- Add a "Restore Purchases" row (case 8) below the Subscription row
- On tap: call `Purchases.shared.restorePurchases()`, then `EntitlementManager.shared.refreshEntitlements()`
- Show brief HUD/alert on success or failure

### 2. Add Manage Subscription link to SettingsViewController
- Add a "Manage Subscription" row (case 9) below Restore Purchases
- Only visible when `EntitlementManager.shared.isPremium == true`
- On tap: open `https://apps.apple.com/account/subscriptions` via `UIApplication.shared.open()`

### 3. Implement Watch Connectivity for entitlement sync
- Create `WatchSyncManager.swift` in `WoofTalk/` — a singleton that adopts `WCSessionDelegate`
- On `EntitlementManager.shared.$isPremium` change: send dictionary `["isPremium": bool, "subscriptionTier": string, "isTrialActive": bool]` via `WCSession.default.updateApplicationContext()`
- On Watch side (`InterfaceController`): implement `session(_:didReceiveApplicationContext:)` to update UI from received entitlement dict
- Activate session in `application(_:didFinishLaunchingWithOptions:)` in AppDelegate if `WCSession.isSupported()`

### 4. Ensure logIn(auth.uid) on every launch
- Already implemented: `RevenueCatManager.observeAuthState()` calls `Purchases.shared.logIn(userId)` on auth
- No changes needed — verify existing code handles this

## Files to Modify

- `WoofTalk/SettingsViewController.swift` — add restore + manage rows
- `WoofTalk/AppDelegate.swift` — activate WCSession on launch
- `WoofTalk/WatchKitExtension/InterfaceController.swift` — receive Watch Connectivity context

## Files to Create

- `WoofTalk/WatchSyncManager.swift` — WCSession delegate + entitlement relay

## Verification

1. Restore purchases button calls RevenueCat API and refreshes entitlements
2. Manage Subscription link opens App Store subscription page
3. Watch UI updates within 5 seconds of iOS entitlement change
4. `logIn(auth.uid)` fires on every app launch (existing behavior)
