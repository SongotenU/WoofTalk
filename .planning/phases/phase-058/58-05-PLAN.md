# Plan 58-05: Add deployment automation

## Task
Create GitHub Actions workflows for release builds and staging deployments.

## Implementation
- File: `.github/workflows/release-build.yml`
  - Trigger: version tags (v*)
  - Jobs: create-release, build-ios-release, build-android-release
  - Creates GitHub Release with changelog
  - Builds and uploads IPA/AAB artifacts

- File: `.github/workflows/staging-deploy.yml`
  - Trigger: push to develop branch
  - Jobs: deploy-web-staging, deploy-ios-testflight, deploy-android-internal
  - Note: iOS/Android deploy jobs disabled by default (require API keys)
