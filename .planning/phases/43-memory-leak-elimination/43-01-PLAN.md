## Must Haves

### Truths (must be true for this plan to succeed)
- All NotificationCenter observers have corresponding `deinit { NotificationCenter.default.removeObserver(self) }`
- All Timer instances are stored as instance properties
- All Timer instances are invalidated in deinit or on completion
- All Timer closures use `[weak self]` to prevent retain cycles
- `LeaderboardManager` has `@MainActor` isolation

### Artifacts

- `WoofTalk/LeaderboardManager.swift` — Timer stored, [weak self], invalidate(), @MainActor
