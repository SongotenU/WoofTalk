# Plan 59-02: Create App Store Screenshots

## Goal
Generate required screenshots for App Store using iOS Simulator.

## Required screenshot sizes
- iPhone 6.5" (1242 x 2688 px): iPhone 14 Plus, 13 Pro Max, 12 Pro Max
- iPhone 5.5" (1242 x 2208 px): iPhone 8 Plus
- iPad Pro 12.9" (2048 x 2732 px): iPad Pro (3rd gen+)
- iPad Pro 11" (1668 x 2388 px): iPad Pro (1st gen+)

## Success Criteria
- [ ] Launch app in iOS Simulator (iPhone 14 Pro Max)
- [ ] Capture screenshots of main screens (5-10 screens)
- [ ] Resize screenshots to required dimensions
- [ ] Save screenshots to `store-assets/screenshots/ios/`
