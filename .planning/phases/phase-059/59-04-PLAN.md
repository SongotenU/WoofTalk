# Plan 59-04: Submit for Review

## Goal
Archive the iOS app in Xcode and upload to App Store Connect for review.

## Pre-requisites
- Xcode project builds successfully (Phase 55 ✅)
- App Store Connect configured (59-03 ✅)
- Metadata prepared (59-01 ✅)
- Screenshots ready (59-02 ✅)

## Steps
1. Open Xcode project: `open /Users/vandopha/Downloads/PersonalSideProjects/WoofTalk/WoofTalk.xcodeproj`
2. Set build target to "Any iOS Device (arm64)"
3. Product → Archive
4. In Organizer window, click "Distribute App"
5. Select "App Store Connect" → "Upload"
6. Choose signing certificate and provisioning profile
7. Upload to App Store Connect

## Success Criteria
- [ ] Archive created successfully
- [ ] Upload to App Store Connect complete
- [ ] Build appears in App Store Connect

## Notes
- Requires Xcode 16.2+ with valid Apple ID signed in
- Archive process takes 5-15 minutes
- Upload requires stable internet connection
