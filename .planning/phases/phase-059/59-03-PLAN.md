# Plan 59-03: Configure App Store Connect

## Goal
Configure App Store Connect with correct bundle ID, certificates, and provisioning profiles.

## Steps
1. Register bundle ID: com.wooftalk.app (if not already registered)
2. Create App ID with required capabilities:
   - In-App Purchase (for subscriptions)
   - Push Notifications (optional, for future features)
3. Create distribution certificate
4. Create provisioning profile (App Store distribution)
5. Configure in Xcode project

## Success Criteria
- [ ] Bundle ID registered: com.wooftalk.app
- [ ] Distribution certificate created
- [ ] Provisioning profile active
- [ ] Xcode project signed correctly

## Notes
- Requires Apple Developer Program membership ($99/year)
- Certificate must be installed in Keychain
- Xcode automatic signing can handle provisioning profiles
