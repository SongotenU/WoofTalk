# Key types and contracts from research

From 40-RESEARCH.md -- DogAvatarController pattern:
```csharp
public class DogAvatarController : MonoBehaviour {
    private Animator _animator;
    void Awake() => _animator = GetComponent<Animator>();
    public void PlayBark() => _animator.SetTrigger("Bark");
    public void PlayHeadTurn(float direction) => _animator.SetFloat("HeadTurnDirection", direction);
}
```

       - Public method PlayBark() that calls _animator.SetTrigger("Bark")
       - Public method PlayHeadTurn(float direction) that calls _animator.SetFloat("HeadTurnDirection", direction) where direction is -1.0f (left) or 1.0f (right)

       - Default state: "DogIdle"
       - State "DogIdle" with AnimationClip "DogIdle.anim"
       - State "DogBark" with AnimationClip "DogBark.anim" and exit time = 1.0 (plays once, returns to idle)
       - State "DogHeadTurn" with AnimationClip "DogHeadTurn.anim" and exit time = 0.7 (early return to idle)
       - Transition Idle -> Bark: trigger parameter "Bark"
       - Transition Idle -> HeadTurn: float parameter "HeadTurnDirection" > 0.5 or < -0.5
       - Transitions Bark -> Idle and HeadTurn -> Idle: hasExitTime = true

    C. Create vr-quest/Assets/Animations/Dog/DogIdle.anim as an AnimationClip YAML:
       - Default curve: a subtle scale oscillation on Y axis (breathing effect), amplitude 0.01, period 2s
       - Duration: 2.0 seconds
       - LoopTime: true

    D. Create vr-quest/Assets/Animations/Dog/DogBark.anim as an AnimationClip YAML:
       - Curve: head moves forward (localPosition.z +0.1) then back over 0.5s

       - Duration: 0.6 seconds
       - LoopTime: false (one-shot)

    E. Create vr-quest/Assets/Animations/Dog/DogHeadTurn.anim as an AnimationClip YAML:
