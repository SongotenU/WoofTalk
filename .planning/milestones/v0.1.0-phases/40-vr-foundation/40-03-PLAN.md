- TFLite model loading on main thread (use async/await during scene load)
- Microphone API high latency on Quest (use OnAudioFilterRead callback)
- Missing AudioListener on OVRCameraRig (must add explicitly)
- Oculus Spatializer not enabled in Project Settings > Audio > Spatializer Plugin

       - Private const string ModelName = "woof_bark_model"
       - Private static string[] ClassLabels = new[] { "bark", "howl", "whine", "silence" }
       - Public async Task<BarkClassifier> LoadModelAsync():
         - Load model file via Resources.Load<TextAsset>(ModelName) or native TFLite/Barracuda loader
         - Initialize interpreter (create Interpreter instance with model bytes)
         - Allocate tensors via interpreter.AllocateTensors()
         - Return new BarkClassifier(interpreter, ClassLabels)
       - If model file not found or load fails, return a mock classifier that always returns ("silence", 0.0f) with warning log
       - Log: "[TFLiteModelLoader] Model loaded: OK bytes, tensors allocated" or "[TFLiteModelLoader] FAILED, using mock"

    B. Create vr-quest/Assets/Scripts/Bark/BarkClassifier.cs:
       - Class BarkClassifier (not MonoBehaviour -- plain C# class)
       - Constructor: BarkClassifier(Interpreter interpreter, string[] classLabels)
       - Private Interpreter _interpreter and string[] _classLabels
       - Public (string className, float confidence) Classify(float[] audioSamples):
         - Input validation: if audioSamples.Length != 1024, pad or truncate to 1024
         - Copy audioSamples to interpreter input tensor: _interpreter.GetInputTensor().CopyFromData(audioSamples)
         - _interpreter.Invoke()
         - Read output tensor: float[] output = _interpreter.GetOutputTensor().CopyToFloatArray()
         - Find max confidence index: int maxIdx = Array.IndexOf(output, output.Max())
         - Return (_classLabels[maxIdx], output[maxIdx])
       - Public void Dispose(): _interpreter?.Dispose()
       - Note: Interpreter type is from TensorFlowLite namespace (or Unity.Barracuda if using Barracuda path)
       - Use "using System;" and "using System.Linq;"
