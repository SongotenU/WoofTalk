<task type="auto">
  <name>Task 1: Create Supabase integration — SupabaseManager, TranslationSync, SettingsSync (X-01, X-02)</name>
  <files>
    - vr-quest/Assets/Scripts/Supabase/SupabaseManager.cs
    - vr-quest/Assets/Scripts/Supabase/TranslationSync.cs
    - vr-quest/Assets/Scripts/Supabase/SettingsSync.cs
    - vr-quest/Packages/manifest.json
  </files>
  <read_first>
    - vr-quest/Packages/manifest.json (existing dependencies)
  </read_first>
  <action>
    A. Create SupabaseManager.cs:
       - MonoBehaviour singleton
       - [SerializeField] private string supabaseUrl
       - [SerializeField] private string anonKey (reference to .env or config file, comment: "TODO: Use secure storage for production")
       - Public event Action<Session> AuthStateChanged
       - Public async Task Initialize(): initialize Supabase client, load session from PlayerPrefs if exists
       - Public async Task SignInAnonymously(): create anonymous session, cache to PlayerPrefs
       - Public async Task SignOut(): clear session

    B. Create TranslationSync.cs:
       - Public async Task SaveTranslation(string originalText, string translatedText, string sourceLanguage, string targetLanguage, string platform, Vector3 spatialPosition):
         - Insert into translation_history table with columns: original_text, translated_text, source_language, target_language, platform ("vr_quest"), spatial_position (JSON: {x, y, z}), created_at
       - Public async Task<List<TranslationRecord>> GetRecentTranslations(int limit = 50):
         - Query translation_history WHERE user_id = currentUser ORDER BY created_at DESC LIMIT limit
       - Public event Action OnSyncComplete

    C. Add Supabase Unity SDK dependency to Packages/manifest.json:
       - "com.supabase.unity": "1.0.0" (or appropriate version)

    D. Create SettingsSync.cs:
       - Public async Task SaveUserSettings(string key, string value):
         - Upsert into user_settings table: {user_id, key, value, updated_at}
       - Public async Task<string> GetUserSettings(string key):
         - SELECT value FROM user_settings WHERE user_id = currentUser AND key = key
       - Public async Task<Dictionary<string, string>> GetAllSettings():
         - SELECT key, value FROM user_settings WHERE user_id = currentUser
  </action>
  <acceptance_criteria>
    - SupabaseManager.cs has Initialize, SignInAnonymously, AuthStateChanged
    - TranslationSync.cs has SaveTranslation with all required fields including platform and spatialPosition
    - SettingsSync.cs has SaveUserSettings, GetUserSettings, GetAllSettings
    - manifest.json includes Supabase dependency
  </acceptance_criteria>
